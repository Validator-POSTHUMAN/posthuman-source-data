# Celestia Mocha Testnet Decentralization Map

Interactive map for public Celestia Mocha testnet RPC and consensus peer infrastructure tracked by POSTHUMAN.

The map shows:

- POSTHUMAN and chain-registry public testnet RPC endpoint health and DNS location;
- public testnet seeds and persistent peers from the Cosmos chain-registry;
- globally routable peers observed through public Celestia Mocha testnet RPC `/net_info`;
- country, provider/ASN and status distribution.

Data feed: `celestia-testnet/decentralize-map.json`.

> Private validator IPs, sentry topology, non-global addresses, and local non-Mocha addrbooks are not published. Only public Celestia Mocha testnet endpoints are shown.
