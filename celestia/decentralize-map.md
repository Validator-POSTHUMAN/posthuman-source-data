# Celestia Mainnet Decentralization Map

Interactive map for public Celestia mainnet RPC and consensus peer infrastructure tracked by POSTHUMAN.

The map shows:

- POSTHUMAN and chain-registry public RPC endpoint health and DNS location;
- public seeds and persistent peers from the Cosmos chain-registry;
- globally routable peers observed through public Celestia mainnet RPC `/net_info`;
- country, provider/ASN and status distribution.

Data feed: `celestia/decentralize-map.json`.

> Private validator IPs, sentry topology, non-global addresses, and local non-mainnet addrbooks are not published. Only public Celestia mainnet endpoints are shown.
