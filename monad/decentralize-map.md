# Monad Mainnet Decentralization Map

Interactive map for public Monad mainnet infrastructure tracked by Posthuman.

The map shows:

- official Monad bootstrap peers from the public node configuration;
- public RPC endpoint health and location;
- country, provider/ASN and status distribution.

Data feed: `monad/decentralize-map.json`.

> Private validator IPs, sentry topology and internal logs are not published. Only public endpoints and official public bootstrap peers are shown.
