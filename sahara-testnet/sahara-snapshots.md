# Sahara AI Testnet — Snapshots

## Snapshot Availability

**Status**: No public snapshots available yet.

Sahara AI testnet is a new network and public snapshot services have not yet added support.

## Alternative Sync Methods

### 1. State Sync (Recommended)

State sync is the fastest way to sync a new node. It downloads a recent snapshot directly from peers.

See the [Installation Guide](installation-guide.md) for state sync configuration.

### 2. Regular Block Sync

If state sync fails, the node will sync from genesis block by block. This is slower but reliable.
