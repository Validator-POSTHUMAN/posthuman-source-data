# Monad Testnet Decentralization Map

Interactive map for public Monad testnet infrastructure tracked by Posthuman.

The map shows:

- official Monad testnet bootstrap peers from the public node configuration;
- public RPC gateway health and Cloudflare edge visibility;
- country, provider/ASN and status distribution.

Data feed: `monad-testnet/decentralize-map.json`.

> Private validator IPs, sentry topology and internal logs are not published. Only public endpoints and official public bootstrap peers are shown.
